# Project Setup

## Material

`ng add @angular/material`

`npm i @angular/material-moment-adapter`

`npm i moment-timezone`

## Firebase

`npm i firebase`

`npm i -g firebase-tools`

`firebase login`

`firebase init hosting`

`firebase init firestore`

`firebase init functions`

## Firebase UI

`npm i firebaseui`

## Tailwind

`npm i tailwindcss`
